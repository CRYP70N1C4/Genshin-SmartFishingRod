## 原神钓鱼辅助工具（无需管理员权限版本）

+ 演示

![](images/demo1.gif)

+ 原理

  采用ESP32充当蓝牙鼠标来发送点击事件

  使用串口给ESP32发送数据

+ 注意事项

  请先配置打包的main.exe的位置

  ![](images/setting.png)

+ 使用方法

  ![](images/guide.png)

  1. 刷入程序
  2. 连接至电脑
  3. 用蓝牙连接该设备
  4. 选择串口号
  5. 打开串口
  6. 点击start
  7. 尽享钓鱼
